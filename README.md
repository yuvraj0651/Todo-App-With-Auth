# ✅ Todo App with Authentication

## 📌 Overview
A task management application with authentication that allows users to securely manage their personal tasks.

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS

## ✨ Features
- 🔐 User authentication
- 📝 Add, update, delete tasks
- 📦 Persistent state management
- 📱 Responsive UI

## 🧠 Key Learnings
- Authentication flow
- State persistence
- User-specific data handling

## 🚀 Future Improvements
- Task categories
- Notifications/reminders
