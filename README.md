# ✅ Todo App with Authentication 🚀

## 📌 Overview
A modern Todo application with user authentication, allowing users to securely manage their tasks with features like create, update, delete, and persistent state handling. Built with a focus on clean UI, scalability, and real-world user flows.

---

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS
- JavaScript (ES6+)

---

## ✨ Features
- 🔐 User authentication (Login / Register)
- 📝 Create, update, and delete tasks (CRUD)
- ✅ Mark tasks as completed / pending
- 🔍 Filter tasks (All / Completed / Pending)
- 💾 Persistent data storage (API / Firebase)
- 🔄 Real-time UI updates with Redux Toolkit
- 🔒 Protected routes for authenticated users
- 📱 Fully responsive design
- 🌙 Dark mode support

---

## 🧠 Key Learnings
- Implementing authentication flows in frontend apps
- Managing protected routes and user sessions
- Handling CRUD operations with global state
- Designing clean and user-friendly task management UI
- Structuring scalable React + Redux applications
